Web Technologies :

WWW ? : World Wide Web  (W3C.Org)
 Network of Information
 - A Uniform naming Scheme for locating a resource on the web (URI)
 - Protocol, for accessing the named resources over web (HTTP)
 - Content / Hyper Text, for easy navigation among resources (HTML)


Resource : (document, images, audio, video, applications, programs etc..) , will have an address that can be accessed using the URI

Eg : https://www.mysite.com/mypage
       1       2             3

1. Protocol, 2 : Machine name, 3. Actual Resource

URI > (URN   || URL)
Uniform Resource name (defines an item identity)
Uniform Resource Locator (provides methods to find the resource)


Web page : a document

HTMl + CSS + JavaScript = Content + Style + Behaviour

HTML : Hyper Text Mark Up Lang. 
       It is known as a publishing lang. of www
       Made up of predefined tags, elements, attributes and values

HTTP : Hyper text Transfer Protocol
      is a network protocol used to deliver virtually any resource on the web
      is an HTTP client, as it sends/receives requests to and from the server
      operates on a default port 8080, to listen to requests


CSS : Cascading Style Sheets
   3 types :
     inline
     internal
     external
can be applied as selectors, properties and values
  -- Tag selectors
  -- ID selectors
  -- Class selectors

Grouping and Nesting :
 grouping : h2{color:red;} .mycolor{color:red} #color{color:red} 
            h2,.mycolor,#color{color:red}

Nesting : #t{background-color:blue; padding :1em'}
          #t h1{color:green;}
          #t p{color:crimson;font-size:20px;}

shorthand properties : border-top :10px; border-right:5px;
        10px 5px 10px 5 px
